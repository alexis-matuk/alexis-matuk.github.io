Dark Mode

Isolated Encounter Shiny Hunting Optimizer

    Find Efficiency of Pokemon
  1. Pick a pokemon from the dropdown menu, that will show you each of the spawning areas where that pokemon is present.
  2. Each area will have a rank and a color gradient, 1 and green being the best.
  3. Hover an area to see each of the pokemon's types and a percentage associated with them. That percentage tells you how efficient it is to hunt that pokemon by using a sandwich of that type in that area.
  4. Craft a lvl 3 sandwich and go around the selected area in-game to look for the spawns

    Find Efficiency of Type (beta)
  1. Pick a type from the dropdown menu.
  2. The map will show every area that has one pokemon with 100% efficiency for the type you picked.
  3. This is most useful if you want to target a type for multiple pokemon during one sandwich.

All of the raw data was obtained from https://www.serebii.net, used for calculations and redrawn on their map.
